## 📌 Project Overview
This project implements various CPU scheduling algorithms in C and visualizes their performance using Python.

## ⚙️ Algorithms Implemented
- FCFS
- SJF (Non-Preemptive)
- SRTN
- Round Robin
- Priority (Preemptive & Non-Preemptive)

## 📊 Features
- Calculates CT, TAT, WT
- Stores output in file
- Python-based visualization:
  - Time analysis graphs
  - Completion timeline
  - Algorithm comparison

## 🛠️ Technologies Used
- C
- Python (Matplotlib)

## ▶️ How to Run
1. Compile and run scheduler.c
2. Choose algorithms
3. Run visualize.py

## 📈 Output
Graphs comparing scheduling algorithms performance
